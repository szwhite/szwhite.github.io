# OSS以及配置Typora 图片上传

目的：实现使用Typora写文档时 插入图片自动上传到阿里云OSS对象存储服务 实现网络访问

OSS文档链接：https://help.aliyun.com/document_detail/31883.html?spm=a2c6h.13066369.0.0.5e1d2c60Y1KTPf&source=5176.11533457&userCode=28kqeewo

1）下载PicGo 软件 下载链接：https://github.com/Molunerfinn/PicGo/releases

2）购买阿里云OSS对象存储服务 创建Bucket 权限为公共读

3）点击个人头像，生成keyid和keysecret 使用子用户 创建用户

4）给添加的用户添加权限

5）创建AccessKey

6）设置PicGo

KeyId,KeySecret 为创建的AccessKey

存储空间名为 Bucket的名称

存储区域为 oss-cn-shenzhen (shenzhen服务器地址)

点击确定，然后点击设为默认图床

7）大功告成，打开Typora测试 插入图片自动上传到阿里云OSS对象存储
