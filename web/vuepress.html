# VuePress

# 快速上手

1.创建并进入新目录

mkdir vuepressLearn && cd vuepressLearn
1

2.初始化项目

npm init
1

3.安装 VuePress 本地依赖

npm install -D vuepress
1

4.创建一个文档

mkdir docs && echo '# Hello VuePress' > docs/README.md
1

1）创建docs/.vuepress 文件夹

2）在.vuepress文件夹下创建public文件夹用于存储公共图片资源

3）在.vuepress文件夹下创建config.js配置文件

4）在config.js文件中添加基础配置

module.exports = {
  title: '白白',
  description: '学习笔记',
  head: [ // 注入到当前页面的 HTML <head> 中的标签
    ['link', { rel: 'icon', href: '/logo.jpg' }], // 增加一个自定义的 favicon(网页标签的图标)
  ],
  base: '/', // 这是部署到github相关的配置
  markdown: {
    lineNumbers: true // 代码块显示行号
  }
};

7.安装热加载插件 实现实时编译

8.配置首页 在docs/README.md文件中添加

---
home: true # 标记此页面是否为首页
heroImage: /logo.jpg #图片资源储存于.vuepress/public 下
lang: zh-CN # 表示本页面的语言为zh-CN(简体中文)
heroText: 白白 # 首页的标题内容
actionText: 点击进入 → # 首页跳转按钮的内容
actionLink: /web/
features: #  表明首页的特征，固定的格式为title + details，以三栏流式布局的方式展示
  - title: 简洁至上
    details: 以 Markdown 为中心的项目结构，以最少的配置帮助你专注于写作。
  - title: 博客
    details: 专注学习，记录点点滴滴
  - title: 关于我
    details: QQ:1029057784
footer: white | Copyright © 2021-present Evan You # 为底部内容，与普通的网页一样，我们可以在footer里面写版权信息
---
